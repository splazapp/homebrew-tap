# Splazapp Homebrew Tap

Homebrew formulae for [Splazapp](https://github.com/splazapp) projects.

## Install

```bash
brew tap splazapp/tap
```

## Formulae

### claudecode-notification

Native macOS notifications for Claude Code with click-to-return.

```bash
brew install splazapp/tap/claudecode-notification
```

See [claude-code-notification](https://github.com/splazapp/claude-code-notification) for details.
